# LostThing
【SpringMVC】——微信小程序“失物招领”后端
项目简介：该系统用于校内的失物招领，有效得帮助师生及时找到失物；项目分为失物模块和拾物模块，都基于登记，查看两大功能。
技术栈 ：SpringMVC，JWT
项目亮点 ：
	( 1 ) 对图片进行了上传时压缩，减轻前端显示压力；
	( 2 )  接口使用了拦截器来进行请求的拦截 ，将需要先登陆才能请求的接口进行拦截判断 ，大大提高了接口的安全性；



功能模块：
（1）失物模块——登记、查看、筛选
（2）拾物模块——登记、查看、筛选
